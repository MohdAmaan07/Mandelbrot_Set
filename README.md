## 🌀 Mandelbrot Set Visualizer

Interactive Mandelbrot Set explorer built with Python and Pygame.

### ⚙️ Requirements

* Python 3.7+
* `pygame`
* `numpy`

Install with:

```bash
uv sync
```

### ▶️ Run

```bash
uv run main.py
```

### 🕹 Controls

* **Zoom In/Out**: Mouse Wheel
* **Pan**: Click & Drag
* **Exit**: `Esc` key
